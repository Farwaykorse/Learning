## Favourites
<!----------------------------------------------------------------------------->
Listing favourite commands as a cheatsheet.
Note that some of these are custom commands added to the git configuration.
These are marked with `(!)`
``````
    git commit --amend
    git checkout -b <name>
(!) git glog
    gitk --all
    git rebase -i <loc>^
    git gc --auto
``````
